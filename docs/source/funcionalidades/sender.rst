ISender
=======

Descripción de que es
---------------------

El "ISender" es el elemento principal que usarmos en este framework para poder procesas las "requests" de nuestro código.

Es una implementación del patrón "Mediator" (Información en ingles `acá <https://www.dofactory.com/net/mediator-design-pattern>`_, en español, `acá <https://geeks.ms/jorge/2020/10/14/mediator-pattern-en-c/>`_), el cual tiene la responsabilidad de enrutar nuestras "requests" a su respectivo "handler".
